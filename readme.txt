== Description ==

This plug-in can also be selected in non-administrator rights group that can be set Biz Calendar.
Specifically , it gives you a display Biz Calendar's Menu and the authority of "manage_options" set to the selected authority group.
It has become the authority that allows the "Settings" in menu.
But, since it is separate from the right to display the menu, that can not be used settings other than Biz Calendar.

== Installation ==

Installation using "Add New Plugin"
1. From your Admin UI (Dashboard), go to Plugins. Click the "Add New" button.
2. Search for plugin, or click the "Upload" button, and then select the plugin you want to install.
3. Click the "Install Now" button.

Manual installation
1. Upload the "biz-calendar-grant" folder to the "/wp-content/plugins/" directory.
